# AGENTS

This project is scaffolded for Vite + React using TypeScript, with source code under `src/ui`.

When responding to user prompts, correct the user's prompt and highlight each error in **bold**.

Each React component must have an associated style file with the same base name, such as `DataGridEx1.css`.

Example scope:
- `DataGridEx1`: basic sortable React Data Grid.
- `DataGridEx2`: grouped task/subtask React Data Grid with expandable task rows.
- `DataGridEx3`: sortable React Data Grid with column filters.
